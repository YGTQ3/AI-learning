在 Supabase Dashboard > Authentication > Settings 中配置：

1. **Site URL**: http://localhost:3000 (开发环境)
2. **Redirect URLs**: 
   - http://localhost:3000
   - https://your-domain.com (生产环境)

3. **Email Templates**: 
   - 自定义确认邮件模板
   - 自定义密码重置模板

4. **Providers**: 
   - 启用 Email provider
   - 可选：启用 Google, GitHub 等第三方登录
